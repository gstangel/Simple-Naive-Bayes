Use "python NaiveBayes.py" to run the program
You will be presented with the following menu
    1. Load a Metadata File
    2. Load Training Data
    3. Run Testing Data
    4. Reset Classifier
    5. Exit
You must load files in the following order:
    1.metadata
    2.train data
    3.test data

You will have the ability to output the results of the test.  The program will append the result at the end of every test instance
